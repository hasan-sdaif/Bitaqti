إعداد Google Apps Script Webhook (للكتابة في Google Sheets)
مطلوب إعداد لمرة واحدة فقط — للسماح بتعديل Google Sheets مباشرة من لوحة التحكم (إضافة/تعديل/حذف عملاء):
  1. افتح Google Sheet الخاص بك
  2. القائمة: Extensions → Apps Script
  3. الصق الكود من الصندوق بالأسفل (استبدل كل ما فيه)
  4. احفظ (Ctrl+S) ثم: Project Settings → Script properties
  5. أضف خاصية: ADMIN_PASSWORD = <نفس INVOICE_PASSWORD>
  6. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
  7. انسخ الـ URL (ينتهي بـ /exec)
  8. في Netlify: أضف متغير SHEETS_WEBHOOK_URL = ذلك الـ URL
📋 اضغط لعرض كود Apps Script الكامل (انسخه كاملاً)
