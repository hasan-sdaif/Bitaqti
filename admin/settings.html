إعداد Google Apps Script Webhook

للسماح بتعديل Google Sheets مباشرة من لوحة التحكم (إضافة/تعديل/حذف)
إعداد لمرة واحدة فقط — اتبع الخطوات التالية بدقة:
  1. افتح Google Sheet الخاص بك
  2. القائمة: Extensions → Apps Script
  3. الصق الكود من الصندوق بالأسفل (استبدل كل ما فيه)
  4. احفظ (Ctrl+S)
  5. Project Settings (⚙️) → Script properties → Add property:
    • المفتاح: ADMIN_PASSWORD
    • القيمة: <نفس INVOICE_PASSWORD>
  6. Deploy → New deployment:
    • Type: Web app
    • Execute as: Me
    • Who has access: Anyone
  7. انسخ الـ Web App URL (ينتهي بـ /exec)
  8. في Netlify: أضف متغير SHEETS_WEBHOOK_URL = ذلك الـ URL
